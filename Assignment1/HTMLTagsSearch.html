b tag

The b HTML element is used to make the text special and to be more attractive. It is an paired tag

i tag

The i HTML element represents a range of text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxonomical designations, among others

u tag

The u HTML element is used to underline the perticular think in any text or to any kind of heading

select tag

The select HTML element is used to put the multiple options in on lable for selecting only one amoung those.

button tag

The button HTML element is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology

hr tag

The hr HTML element represents a thematic break between paragraph-level elements: for example, a change of scene in a story

br tag

The br HTML element produces a line break in text
